


## 错题1
规定标准乒乓球的重量应为 2.4 克。
A、B、C、D、E 五个乒乓球，与标准重量比较后可表示为：
A：+0.04 克
B：−0.25 克
C：0 克
D：+0.01 克
E：−0.1 克
求这 5 个乒乓球的平均重量。


思路一：先求每个球的实际重量

标准重量是 2.4 克
所以：
A：2.4 + 0.04 = 2.44 克
B：2.4 − 0.25 = 2.15 克
C：2.4 + 0 = 2.40 克
D：2.4 + 0.01 = 2.41 克
E：2.4 − 0.10 = 2.30 克

(2.44 + 2.15 + 2.40 + 2.41 + 2.30) ÷ 5
= 11.70 ÷ 5
= 2.34 克

思路二：先求“与标准的平均差”
0.04 - 0.25 + 0 + 0.01 - 0.10 = -0.30
-0.30 ÷ 5 = -0.06
2.4 - 0.06 = 2.34 克



## 错题2
一辆汽车 2.5 小时行 320 千米
照此计算，再行 1.5 小时
汽车共行多少千米？


* 学生答案
320 ÷ 2.5 = 128 (km/h)
128 × 1.5 = 189  
320 + 189 = 509 km

* 正确答案
128 × 1.5
= 128 × (3/2)
= 192

* 错误原因
128 × 1.5 计算为 189
