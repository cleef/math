# cube-surface-lab

立方体表面积学习应用（剪边、展开、主轴观察）。

## 功能
- 在立方体 12 条边中点选要剪开的边。
- 使用“模拟剪刀”逐条动画剪开。
- 剪完后自动判定是否可完整展开，成功时自动展开为平面。
- 支持手动展开、重新折叠，观察正方体与展开图之间的关系。
- 第 2 页提供 11 种正方体展开图，目标剪边按钮可直接触发剪刀动画。
- 第 3 页观察前 6 种 `1 + 4 + 1` 展开图，找出 4 个正方形组成的主轴，并理解剩余 2 个面如何封闭正方体。

## 本地运行

```bash
cd light-learning-apps/apps/cube-surface-lab
npm install
npm run dev
```
