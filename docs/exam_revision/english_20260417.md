## Tricky points

| 类型    | 例子           | 本质 |
| ----- | ------------ | -- |
| be动词  | was / were   | 过去状态 |
| 一般动词  | did / had / played | 过去动作 |
| 不规则变化 | became       | 过去变化 |
| 结果动词  | made         | 过去产出 |
